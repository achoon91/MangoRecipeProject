# MangoRecipeProject
ThirdTeamProject
프로젝트 소개

프로젝트명 : MangoRecipeProject
중략 : Spring 기반의 음식(맛집,레시피,최저가,마트정보) 사이트 구현
주요 기능
1. 회원가입/ 로그인
2. 게시판 글 추가,삭제,수정,댓글을 프로시저로 구현
3. 식재료를 이용한 나만의 레시피 추천
4. 맛집 추천
5. 레시피 추천
6. 최저가 추천
7. 날씨별, 기분별 요리 추천

사용 기술
1. 백앤드
언어 : Java,Kotlin
프레임워크 : Spring
웹 서버: Apache
DB : Oracle,MongoDB
주사용기술 : HTML Paser,OpenAPI
1. 프론트앤드
언어 : XML, Javascript
주사용기술 : React, Ajax, MyBatis, OpenAPI, JSP
